### Bolt.Py — AI Gmail Auto-Responder Chrome Extension

Bolt is a lightweight Chrome extension that lives inside your Gmail. With one click, it classifies your email and writes a smart, context-aware reply using AI.

## 🧠 What It Does

📥 Reads the subject and body of the open Gmail email

🧾 Classifies it into predefined labels (like “Documents & Certificates”, “Payment & Fees”, etc.)

✍️ Generates a personalized reply using Google Gemini

💬 Injects the response directly into the Gmail reply box

## ⚙️ How to Use It
⚠️ This extension is not on the Chrome Web Store yet. You’ll install it locally as a developer.

## ✅ Step-by-Step:
Download the extension folder (or clone this repo):
```
git clone https://github.com/Hasan-Shkoukani/bolt.git

Go to chrome://extensions/ in Chrome

Toggle Developer Mode (top-right corner)

Click Load Unpacked

Select the extension folder

Open Gmail, open any email, and you’ll see a new blue button:
“AI Response”
```

Click it. The AI will:

Analyze the subject and body

Talk to the backend

Generate a response

Insert it into the reply box, ready for you to review and send!

## 🛠️ How It Works

The extension fetches:

Subject: from Gmail’s header

Body: from Gmail’s message content

Sends both to a backend Flask API (already hosted)

The API:

Classifies the email using a Hugging Face model

Uses Gemini to generate the reply

The reply is then injected directly into the Gmail composer

## 🧪 Example Use Case
Email:

"I would like to request a certificate of enrollment for the current semester."

Click "AI Response"

## ✅ Reply generated and ready:

“You can request your certificate of enrollment by visiting the registrar's portal...”

## 📝 Notes
Works only on the new Gmail interface

Requires an active internet connection

Responses are generated using Google Gemini Flash

The classification is based on a custom-trained RoBERTa model

## 💡 Built With

Plain JavaScript (no frameworks)

MutationObserver DOM injection

Flask + Hugging Face + Gemini (on backend)

Injected response styled and labeled with:
>>> This response was generated by Bolt.Py

## 🔐 Privacy
This extension does not store or track your emails.

```All processing is done via a secure backend hosted by the developer on render.```
